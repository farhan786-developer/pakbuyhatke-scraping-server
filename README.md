# Pak Buy Pro - Scraping Server

## What This Does
Scrapes PriceOye, Mega, and Daraz for price comparison

## Files
- `app.py` - Scraping server with smart matching
- `requirements.txt` - Dependencies
- `Procfile` - Railway config

## Deploy to Railway
1. Create GitHub repo: pakbuypro-scraping-server
2. Upload these 3 files
3. Railway → New Project → Deploy from GitHub
4. Add Variable: AI_SERVER_URL = [your AI server URL]
5. Generate domain
6. Test: https://your-url.railway.app/health

## Done! ✅
